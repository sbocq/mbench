mbench.benchmark

Config

sealed trait Config[-I, +C, S] extends Serializable

A configuration for a test in a benchmark.

In addition to the input of a benchmark, tests can take a configuration as parameter, see the factory methods in Test or TestSeq. A configuration is either empty, a static configuration C (see StaticConfig), a runtime configuration (see RuntimeConfig) that creates a setup S, or the composition of runtime configuration and a static configuration.

Runtime and static configurations are created using the factory methods in the companion object of this class. Eventually, a runtime configuration can be combined with a static configuration using its and combinator.

See also the factory methods in the companion objects of Test and TestSeq, which make it easy to declare tests that depend on either the input of a benchmark, a runtime and/or a static configurations.

I

The type of the input of a benchmark.

C

The type of the static configuration.

S

The type of setup created by a runtime configuration.
